== Description ==

サイトを公開したのに検索エンジンに表示されないという経験はありませんか？
**Stop the Bokettch**は、検索エンジンがサイトをインデックスできない設定にチェックを入れていた場合、ツールバー内に警告表示を行います。サイト公開時のチェックの外し忘れの防止に役立ちます。
Notice : 警告表示は、投稿者権限グループ（publish_posts権限を持つ権限グループ）で表示されます。

= What's "Bokettch" =
WordBench千葉（WordBenchは、日本のローカルユーザーグループ）で活動しているうぇぶるじょんさん（[@Webourgeon_com](https://twitter.com/Webourgeon_com)）のこと。あるいはその天然なおっちょこちょいな行動の総称をいう。特にうぇぶるじょんさんに認定されたぼけっちーずは番号が振られている。なお、最近ではその様子をヲチるための「[ぼけっち録](http://bokettch.warna.info/)」というサイトが存在するほどである

代表的なぼけっちの例として

* 平らな地面なのに転ぶ
* 迷子になる
* お湯を沸かしすぎて蒸発して少なくなる
* 電車を乗り間違える。違う駅で降りる。
* 炭酸飲料を開けたとき噴水にする。

などが挙げられます。

Special Thanks: [megane9988](http://m-g-n.me/), [Webourgeon_com](http://webourgeon.com/), [nekomimiTaicho](http://nekomimi-taicho.com/)

Translation: [@mamy315](https://twitter.com/mamy315)

== Installation ==

プラグイン一覧ページで、**Stop the Bokettch** を有効化するだけです。
もっと何か作業したかったですか？ゴメンナサイ。これだけです！
